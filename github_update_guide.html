🌐 サイト更新マニュアル

西成住之江支部 ポータルサイト｜担当者向け手順書

https://matsuba-commits.github.io/nishisumi/

📋 更新の全体の流れ（5分でできる）

1
トムに「○○を変更して」と伝える
2
トムが修正したHTMLファイルをダウンロード
3
GitHubを開いてファイルをアップロード
4
「Commit changes」を押す → 完了！

STEP 1 — GitHubを開く

1
GitHubにアクセスしてログイン
ブラウザで以下のURLを開く
https://github.com/matsuba-commits/nishisumi
ブックマークに登録しておくと便利です

STEP 2 — ファイルをアップロード

2
「Add file」→「Upload files」をクリック
画面上部の緑のボタン付近にある「Add file」をクリックして「Upload files」を選ぶ
📸 ［Add file ▼］ボタンの画像
3
修正済みのHTMLファイルをドラッグ＆ドロップ
トムからダウンロードしたHTMLファイルを点線のエリアにドラッグする
📸 ドラッグ＆ドロップエリアの画像
ファイル名が表示されたらアップロード成功
同じファイル名で上書きされる。古いファイルは自動で置き換わる

STEP 3 — 保存して公開

4
「Commit changes」（緑ボタン）を押す
画面下部の緑ボタンをそのまま押すだけでOK。コメント入力は不要。
📸 「Commit changes」ボタンの画像
1〜2分後にサイトに反映されます。反映前はキャッシュをクリアして確認してください（Ctrl＋Shift＋R）

確認用URL

5
サイトを確認する
https://matsuba-commits.github.io/nishisumi/
index.html → トップページ
group.html → 小グループ会ページ
report.html → 活動報告ページ

困ったときは

トムに相談する
文章の変更・追加 → トムに依頼
写真の差し替え → 写真をトムに送る
レイアウトの修正 → トムに依頼
「ここを変えて」と言えばトムがHTMLを修正してファイルを渡します。あとはアップロードするだけです。
